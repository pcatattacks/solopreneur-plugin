# Example: Agency Owner

## Profile
A solo agency owner managing client projects. Handles sales, project management, delivery, and client relationships.

## Employees (Agents)
1. **Account Manager** - Client communication, relationship management, upselling
2. **Creative Director** - Design direction, brand consistency, creative briefs
3. **Developer** - Implementation, technical architecture, code quality
4. **Project Manager** - Timeline tracking, resource allocation, risk management

## SOPs (Skills)
1. `/agency:new-lead` - Qualify an inbound lead and draft a response
2. `/agency:proposal` - Generate a client proposal from a project brief
3. `/agency:brief` - Create a creative/technical brief for a project
4. `/agency:kickoff` - Run a project kickoff (parallel research + planning)
5. `/agency:deliverable` - Review a deliverable before sending to client
6. `/agency:invoice` - Generate an invoice and follow-up email
7. `/agency:retro` - Run a project retrospective

## Team Meetings (Agent Teams)
1. **Sales Sprint**: Account Manager + Creative Director (pitch prep)
2. **Delivery Team**: Developer + Creative Director + Project Manager
3. **Client Review**: Account Manager + Project Manager + Developer

## Tools (MCP Servers)
- Slack (client communication)
- Notion or ClickUp (project management)
- Figma (design files)
- Stripe or FreshBooks (invoicing)
- GitHub (code delivery)

## Lifecycle
`/new-lead` → `/proposal` → `/brief` → `/kickoff` → `/deliverable` → `/invoice` → `/retro`
