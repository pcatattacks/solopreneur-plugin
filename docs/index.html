Claude Code Plugin

Solopreneur

Your Virtual AI Company

A Claude Code plugin that gives you a full AI team — specialized agents, guided workflows, and decision memory. Ship products faster as a team of one.

Open Claude Code in your terminal and run:

1 /plugin marketplace add pcatattacks/solopreneur-plugin
2 /plugin install solopreneur@solopreneur
3 /solopreneur:help
